# Day_21
Multi-Agent Researcher and Writer is an AI-powered system built using AutoGen and Google Gemini. It uses two intelligent agents: a Researcher Agent that collects and organizes information on a given topic, and a Writer Agent that converts the research into a well-structured, informative article.

### Project Description :

The **Multi-Agent Researcher and Writer** project is an AI-based application developed using **AutoGen** and **Google Gemini**. It consists of two specialized AI agents working together to automate content creation. The **Researcher Agent** gathers relevant information, key facts, and important points about a user-provided topic. The **Writer Agent** then analyzes the research and generates a clear, organized, and engaging article with headings and a conclusion. This multi-agent approach divides tasks efficiently, resulting in accurate research and high-quality written content. The project demonstrates how multiple AI agents can collaborate to complete complex tasks more effectively than a single agent.

---

### Features

* Uses multiple AI agents to complete a task.
* Researcher Agent collects and organizes information.
* Writer Agent creates a structured article.
* Generates informative and readable content.
* Accepts any topic as user input.
* Built with Python, AutoGen, and Google Gemini.
* Demonstrates AI agent collaboration and task delegation.
* Easy to run using the command line.
